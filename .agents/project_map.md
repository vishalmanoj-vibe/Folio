## "If You Need To…" Decision Table

| Task | Read first | File to edit |
|------|-----------|-------------|
| **Add a new chart** | `skills/add_chart.md` | `components/charts/<name>.py` + `callbacks/chart_callbacks.py` |
| **Add a new callback** | `docs/reference/callback_ownership.md` — check ID isn't taken | The relevant `callbacks/<page>_callbacks.py` |
| **Touch a dcc.Store** | `docs/reference/store_contracts.md` — never guess the shape | `app.py` (store seed) + owning callback |
| **Touch a chart builder** | `docs/reference/known_issues.md` — check BUG-001, BUG-005 first | `components/charts/<name>.py` |
| **Touch a pattern-matched callback** | `docs/reference/known_issues.md` — check BUG-002 first | The relevant `callbacks/<page>_callbacks.py` |
| **Find a file path** | `skills/surgical_edit.md` — use grep, never scan | `grep -rn 'def func_name' .` |
| **Add a component ID** | `skills/registry.md` — check for collision first | The page/component file, then update registry |
| **Debug a silent crash** | `skills/debug_dash.md` | The callback/chart that's failing |
| **Add a new data fetch** | `skills/data_fetching.md` | `services/market/` |
| **Check who owns an Output ID** | `.agents/generated/callback_index.md` (auto, by line) or `docs/reference/callback_ownership.md` (human, with notes) | — |
| **Check all store IDs** | `.agents/generated/store_index.md` | `app.py` |
| **Understand data flow** | `docs/guides/DEVELOPER_GUIDE.md` | — |
| **Fix a known regression** | `docs/reference/known_issues.md` — find the bug entry, read the fix | The "Files affected" file in that entry |
| **Change technical indicator math** | `skills/technical_analysis.md` | `services/technical_indicators.py` |
| **Style a new component** | `skills/ui_ux.md` + `skills/aura_design_system.md` | The component's CSS file + `assets/` |
| **Write a new test** | `skills/testing.md` | `scratch/tests/` |

## Auto-Generated Files — How to Refresh

Run from the project root:

```bash
python3 .agents/generated/sync_docs.py           # regenerate both index files
python3 .agents/generated/sync_docs.py --dry-run # preview without writing
```

**When to run**: Before starting any build cycle that adds, removes, or renames Output IDs or `dcc.Store` components.

**What it touches**: Only `.agents/generated/callback_index.md` and `.agents/generated/store_index.md`. Human docs are never modified by the script.
